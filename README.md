# FabricTPA Updated

FabricTPA Updated is a fork of the TPA mod used on LuminaCC which itself is a heavily modified version of FabricTPA by HalloTheEngineer. It adds a way for players to teleport to each other

## Features
Adds the following commands:
- `/tpa`
- `/tpahere`
- `/tpaccept`
- `/tpdeny`
- `/tpcancel`

## Changes from LuminaCC's version
- Updated from 1.20.1
- Feedback is sent to the recipient of a request when they accept/deny a request
- `/tpahere` requests are fully functional now
- `/tpahere` requests are considered for outbound requests when sending normal TPA requests

## Changes from HalloTheEngineer's version to LuminaCC's version
- All feedback is sent in the chat
- Mod is in English (done with Google Translate originally then fixed up later)
- Teleporting doesn't require XP to be done
- There is no countdown on TPA requests being done
- There are now buttons on the teleport request message
